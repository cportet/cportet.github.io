### Politique de confidentialité

Dernière mise à jour : 21 novembre 2025

#### 1. Introduction
Les applications publiées par **Cyril Portet Dev** appliquent toutes la même charte de confidentialité.
Ces applications sont développées par un développeur indépendant.  

Cette politique de confidentialité explique quelles données ces applications collectent (ou ne collectent pas), comment elles sont utilisées et vos droits en tant qu’utilisateur.

#### 2. Données collectées
**Les applications développées par Cyril Portet ne collectent, ne stockent ni ne transmettent aucune donnée personnelle.**

Voici précisément ce que peuvent faire ces applications :

- Elles peuvent lire la liste des applications installées sur votre appareil Windows via les API publiques et officielles de Windows (Windows.ApplicationModel.Package).
- Elles peuvent enregistrer localement sur votre appareil vos préférences d’organisation (favoris, groupes, ordre, couleurs, etc.) dans un fichier de configuration stocké uniquement dans le dossier local de l’application (AppData).
- Elles ne contiennent aucun système de télémétrie, aucun tracker, aucun service analytique tiers (pas de Microsoft Clarity, Google Analytics, etc.).
- Elles n’accèdent pas à vos fichiers personnels, photos, contacts, localisation ou toute autre donnée sensible.
- Elles ne nécessitent et n’utilisent aucune connexion Internet.

#### 3. Permissions demandées
Aucune permission spéciale n’est requise.  
Les applications fonctionnent entièrement hors-ligne et en mode sandbox Windows.

#### 4. Partage ou transmission de données
Aucun partage ni transmission de données n’a lieu.  
Vos données restent exclusivement sur votre appareil.

#### 5. Sécurité des données
Toutes les données de configuration sont stockées localement sur votre PC dans le répertoire protégé de ces applications.  
En cas de désinstallation complète via les paramètres Windows ou le Microsoft Store, ces données sont automatiquement supprimées.

#### 6. Conformité RGPD & Microsoft Store
- Les applications respectent pleinement le Règlement Général sur la Protection des Données (RGPD).
- Elles sont déclarées auprès du Microsoft Store dans la catégorie « Aucune donnée collectée ».
- Aucune donnée n’est transférée vers des pays tiers.

#### 7. Modifications de la politique de confidentialité
En cas de mise à jour majeure d’une application entraînant une modification du traitement des données, une nouvelle version de cette politique sera publiée et la date de mise à jour sera modifiée en haut de page.
