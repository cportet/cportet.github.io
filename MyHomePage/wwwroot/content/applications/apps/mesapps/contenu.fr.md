L'application "Mes Applications" était un prototype réalisé avec WinUI pour la réalisation d'un lanceur d'applications universel.
Elle permettait de regrouper et d'organiser toutes vos applications installées sur votre appareil Windows dans une interface unique et personnalisable.

Cette application n'est pour l'instant plus maintenue, mais j'envisage de reprendre prochainement le code, le publier sur GitHub et mettre à jour l'application sur le store.
